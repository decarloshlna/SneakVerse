# SneakVerse 👟

SneakVerse adalah sebuah platform web e-commerce modern yang dirancang khusus untuk para pencinta dan kolektor sneakers. Project ini fokus pada penyediaan antarmuka yang bersih, intuitif, dan responsif untuk menjelajahi serta mengelola katalog produk sepatu terbaru.

## 🚀 Fitur Utama

*   **Katalog Produk Dinamis:** Menampilkan berbagai koleksi sneakers terbaru dengan informasi detail yang lengkap.
*   **Manajemen Autentikasi:** Sistem login dan register yang aman untuk membedakan hak akses pengguna.
*   **Sistem Checkout:** Alur proses simulasi pembelian produk yang terstruktur.
*   **Desain Responsif:** Tampilan yang optimal saat diakses melalui perangkat mobile maupun desktop menggunakan kombinasi HTML dan CSS modern.

## 🛠️ Teknologi yang Digunakan

*   **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
*   **Backend:** PHP (Native)
*   **Database:** MySQL / MariaDB
